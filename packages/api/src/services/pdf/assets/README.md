# Logo Assets

Place your company logo file here as `logo.png`

## Requirements:
- File name: `logo.png`
- Format: PNG (recommended)
- Size: Approximately 150x150 pixels or similar square ratio
- The logo will be displayed at 25x25mm in the PDF header
